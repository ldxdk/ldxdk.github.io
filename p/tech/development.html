Cargo

cargo new [projectname]          # create new project
cargo build                      # build debug build
cargo build --release            # build release build
cargo check                      # check if can build (faster than building)
cargo update                     # update creates
cargo run                        # build and run
cargo doc --open                 # build docs (incl. crates docs) and open
cargo clean                      # cleanup build files

CocoaPods - installation

cd
chown bs:staff .local
curl -L https://get.rvm.io | bash -s stable

Reopen terminal, then:

rvm install ruby-3.1.2
rvm use ruby-3.1.2
rvm --default use 3.1.2
gem update --system
gem install cocoapods -V

NPM: Make packages local to user

mkdir ~/.npm-global
npm config set prefix '~/.npm-global'

... add to profile:

NPM_PACKAGES="$HOME/.npm-global"
PATH="$NPM_PACKAGES/bin:$PATH"
export NPM_PACKAGES PATH
